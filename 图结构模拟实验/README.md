# 图结构模拟实验（Literature-TKG Simulation）

本子目录用于：在**无 CV/ASR 噪声**的前提下，通过“小说/纪传体史书”构造时空知识图谱，用来压测 **TKG-Graph-v1.0-Ultimate** 的设计边界、验证查询模式，并为路演 Demo 提供素材。

建议优先使用：三国（《三国演义》+《三国志》年表）、明史（《明朝那些事儿》）等时间轴相对清晰的材料。

## 文件索引

- `01_建图方法论与工作流.md`  
  - 定义：如何从文本（章节/段落）出发，由 LLM/Agent 生成标准化 JSON 事件集，并逐步收敛到可复用的建图流程。
- `02_JSON_to_TKG_导入算法与Neo4j写入.md`  
  - 定义：如何从上述 JSON 中解析出节点/边，如何做去重、ID 生成、事务写入与幂等控制。
- `03_查询器设计与用例集.md`  
  - 定义：围绕构建好的 Literature-TKG，设计一组标准查询与 Cypher 模板，用于边界测试与 Demo 演示。
