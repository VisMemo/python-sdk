### HTML版 20页结构（参考）

原20页内容通过以下方式精简到15页：

**合并的页面：**
- 市场洞察 + 市场规模 → 整合为"万亿赛道"和"TAM/SAM/SOM"
- 产品形态 + 商业模式 → 合并为"AI陪伴SDK"
- 竞争壁垒 + 避开死亡陷阱 + 竞争分析 → 整合为"竞争矩阵"
- 市场策略(双轨) → 融入"AI陪伴SDK"页面
- 风险管理 → 关键点融入相关页面
- 联系方式 → 合并到封面页和愿景页

**删除/精简的独立页面：**
- 单独的"联系方式"页 → 信息移至封面和愿景页
- "避开死亡陷阱"独立页 → 核心内容融入竞争分析
- "风险管理"独立页 → 关键风险分散到相关页面
- "市场策略(双轨)"独立页 → 整合进产品与商业模式

### 精简原则

1. **投资人视角优先**：保留最能打动投资人的核心内容
2. **故事线流畅**：机会→问题→方案→壁垒→进展→融资
3. **数据可视化**：用图表替代大段文字描述
4. **一页一主题**：每页聚焦单一核心信息点
5. **15分钟演讲**：每页约1分钟，适合路演时长

### 技术实现亮点

- **React 18** + **Vite** 现代化构建
- **Ant Design 5.x** 企业级UI组件
- **@ant-design/charts** G2数据可视化
- **Framer Motion** 流畅动画效果
- **16:9固定布局** 适配所有演示场景
- **响应式设计** 兼容不同屏幕尺寸

### 使用说明
